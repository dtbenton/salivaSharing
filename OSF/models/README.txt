These are the scripts and example files needed to run the simulations reported in

Benton, D.T. and Hoch, J.E. (2026).  An associative learning account of how saliva becomes a cue for comfort.  Child Development. 

The simulations are run using a modified version of the Lens neural network
simulator (http://tedlab.mit.edu/~dr/Lens/).  Binary executables for Windows,
OSX, and Linux are provided at

http://www.cnbc.cmu.edu/~plaut/Lens
